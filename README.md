# Eat Seasonably

Eat Seasonably is a React application focused on seasonal eating.  
The project helps users explore seasonal fruits and vegetables and discover recipe ideas based on products that are currently in season.

## Features

- Browse seasonal fruits and vegetables
- View products by season
- Navigate between pages using React Router
- Clean and user-friendly interface
- Frontend structure built with reusable React components

## Technologies Used

- React
- JavaScript
- React Router
- CSS / LESS
- HTML

## Project Structure

The project is organized into two main folders:

- `client` – React frontend application
- `server` – backend/server-side part of the project
